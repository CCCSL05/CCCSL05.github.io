Redis常用命令

一、连接redis服务命令

  • 连接本地Redis：redis-cli
  • 连接远程服务器的Redis：redis-cli -h host -p port -a password，没有密码可省略 -a选项

三、key命令

Redis，key-value，key作为记录的唯一标识，相当于MySQL的id。

可对该条记录做一些操作。

  1. keys *，查看所有key

  2. del [keyname]，删除key

  3. type [keyname]，查看存储的类型（redis有5种类型）

  4. ttl [keyname]，查看key的过期时间，秒。-1永不过期，-2已经过期（数据库就不存在了）。

    pttl [keyname]，过期时间，毫秒。

  5. expire [keyname] [second]，设置过期时间，秒

  6. persist [keyname]，设置持久化，永不过期。

  7. randomkey，随机返回一个key

  8. flushdb，删除当前数据库的所有key

    flushall，删除所有数据库的key

  9. 1

四、String

set，get，是String类型专属的。

String，<String,String>

  1. set key value，给key设置String类型的值。value加不加 ""都一样，默认会加。

  2. get key，获取String类型key的值

  3. mget key1 key2..，获取多个key的value

  4. mset k1 v1 k2 v2..，设置多个

  5. incr key，key++。必须是int类型（小数不行）

    decr key，key–

  6. incrby key number，key+=number

    decrby key number，key-=number

  7. append key value，字符串追加值，str=str+value

  8. strlen key，返回字符串的长度

  9. setnx key value，【分布式锁】

五、hash

hash，<String，field，value>

Redis hash 是一个 string 类型的 field（字段） 和 value（值） 的映射表，hash 特别适合用于存储对象。

这里的Field与String类似，操作也相似。

  1. hset key field value [f2 v2]，设置一个key的多个field和value。

    hget key field，获取一个key-field的value

  2. hmget key f1 f2，获取多个字段的值

    hmset key f1 v1 f2 v2，设置多个字段的值

  3. hgetall key，获取一个key所有的field和value

    • hkeys key，获取一个key所有的所有field
    • hvals key，获取一个key所有的所有value

  4. hdel key field1 field2，删除多个字段

  5. hlen key，获取字段的数量

  6. hincrby key field num，给字段增加num

六、list

这个不是数组，是队列。

  1. lpush key v1 v2，给数组添加多个值，头插法，先插入v1

    lpushx key v1 v2，头插法，list必须已经存在

    区别：lpush不存在会创建，lpushx不存在会报错

    rpush key v1 v2，

    rpushx key v1 v2，

  2. lrange key 0 -1，查看队列范围元素（0 是第一个，-1是最后一个）

  3. lpop key，弹出队列第一个元素（最后一个插入的），并返回值

    rpop key，弹出最右边的（最先插入的），l，left，r，right

    rpoppush，把最后一个元素放到第一个位置，先pop再push

    lrem key count value，移除count个value

  4. lset key index value，通过下标设置值

  5. ltrim key 0 6，只保留范围内的元素

  6. llen key，查看队列长度

七、set

set，<String>

  1. sadd key member1..，向集合中添加多个元素

  2. srem key member1..，移除元素

  3. smembers key，查看集合的成员

    scard key，查看集合成员数量

  4. sinter k1 k2..，查看交集

  5. sunion k1 k2..，查看并集

  6. sdiff k1 k2 ，查看差集，k1 - k2

  7. sismember key member，判断member是否是该集合的元素

八、zset

排行榜

权重，double类型

  1. zadd key score member1 ...，添加多个成员

  2. zrange key 0 -1 [withscores]，查看范围元素

  3. zcount key min max，统计分数区间的数量

    zcard key，统计全部数量

  4. zscore key member，返回成员的分数

  5. zrevrangebyscore key max min withscores，分数区间的成员，从高到低

其他

  1. select 1，切换到数据库db1
