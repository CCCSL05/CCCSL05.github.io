使用Stream，

  1. 创建Stream流
  2. 操作Stream流
    1. 过程操作
    2. 收尾操作

Stream流保存的是算法，而不是列表（具体的数组）

小结

Stream提供的常用操作有：

转换操作：map()，filter()，sorted()，distinct()；

合并操作：concat()，flatMap()；

并行处理：parallel()；

聚合操作：reduce()，collect()，count()，max()，min()，sum()，average()；

其他操作：allMatch(), anyMatch(), forEach()。

创建Stream

// Stream.of方法
Stream<String> stream = Stream.of("A", "B", "C", "D");

// 数组 或 Collection
Stream<String> stream1 = Arrays.stream(new String[] { "A", "B", "C" });
Stream<String> stream2 = List.of("X", "Y", "Z").stream();

// 简单快速创建无限长度，全是1
IntStream.generate(() -> 1).limit(100).forEach(System.out::println);

基于Supplier<Stream元素类型>，Stream.generate()方法，它需要传入一个Supplier对象，生成一个Stream流。

public class Main {
    public static void main(String[] args) {
        Stream.generate(new LocalDateSupplier())
                .limit(31)
                .filter(ldt -> ldt.getDayOfWeek() == DayOfWeek.SATURDAY || ldt.getDayOfWeek() == DayOfWeek.SUNDAY)
                .forEach(System.out::println);
        
//        Stream.generate(new IntSupplier())
//                .limit(100)
//                .forEach(System.out::println);        
        
    }
}

class LocalDateSupplier implements Supplier<LocalDate> {
    LocalDate start = LocalDate.of(2020, 1, 1);
    int n = -1;
    public LocalDate get() {
        n++;
        return start.plusDays(n);
    }
}

// limit 100
class IntSupplier implements Supplier<Integer> {
    int n = -1;

    public Integer get() {
        n++;
        return n;
    }
}

过程操作

过程操作（或者说叫转换操作），会对当前Stream流进行处理（更改算法规则，所以很快），返回一个新的Stream流。

流是一次性的，操作后会改变，当金星收尾操作后，该流将不复存在。

Map

遍历每一个流元素，映射成另一个元素，最后变成一个新的Stream流。

返回Stream流，可以继续操作。

Stream<Integer> s = Stream.of(1, 2, 3, 4, 5);
Stream<Integer> s2 = s.map(n -> n * n); // n -> n * n 基本写法
// out，1 4 9 16 25 

// 使用方法引用
Stream.of("  Apple ", " pear ", " ORANGE", " BaNaNa ")
    .map(String::trim) // 去空格
    .map(String::toLowerCase) // 变小写
    .forEach(System.out::println); // 打印

filter

过滤出，按照条件是否在Stream流里删除该元素。最后变成一个新的Stream流。

判断为真则留下，为假则删除。

返回Stream流。

IntStream.of(1, 2, 3, 4, 5, 6, 7, 8, 9)
    .filter(n -> n > 5)
    .forEach(System.out::println);
// 6 7 8 9

sorted

对Stream流里的元素进行排序。

返回排序后的Stream流。

Stream.of(1, 7, 8, 9, 6).sorted().forEach(System.out::println);

Stream.of("Orange", "apple", "Banana").sorted().forEach(System.out::println);

skip & limit

跳过几个，输出几个。

返回截取之后的Stream流。

Stream.of(1, 2, 3, 4, 5, 6)
    .skip(2)
    .limit(2)
    .forEach(System.out::println);

distinct

去重

返回去重后的Stream流。

Stream.of(1, 2, 3,1,2,3).distinct().forEach(System.out::println);

concat

连接另一个Stream流，拼接成一个新的Stream流。

返回合并后的Stream流。

Stream<String> s1 = Stream.of("A", "B", "C");
Stream<String> s2 = Stream.of("D", "E");
// 合并:
Stream<String> s = Stream.concat(s1, s2);
System.out.println(s.collect(Collectors.toList()));

parallel & parallelStream

parallel 把Stream流，变成并行流。获取处理元素的顺序不一定了（sorted会失效），增加处理效率。

parallelStream() − 为集合创建并行流。

返回并行的Stream流。

Stream.of(1,2,3)
    .parallel()
    .forEach(System.out::println);
// 2 1 3，输出顺序不是1 2 3，说明并发执行了。

flatMap

啊啊啊

返回Stream流。

Stream<List<Integer>> s = Stream.of(
    Arrays.asList(1, 2, 3),
    Arrays.asList(4, 5, 6),
    Arrays.asList(7, 8, 9));
s.flatMap(list -> list.stream())
    .forEach(System.out::println);

peek

偷看。相当于可以返回stream流的foreach。

返回stream流。

myusers.stream()
    .peek(myuser -> myuser.setId(myuser.getId() + 100))
    .forEach(System.out::println);

收尾操作

会返回一个结果值。

forEach

遍历Stream流的每一个元素，相当于for循环。

没有返回值。

Stream<Integer> s = Stream.of(1, 2, 3, 4, 52);

s.forEach(System.out::println); // 方法引用，可以看一下println方法的参数和返回

s.forEach(obj -> System.out.println(String.valueOf(obj))); // 一般写法，与上面效果相同

s.forEach((obj) -> {
    // 因为指定了泛型为Integer，所以可以确定obj为Integer类型。所以可以对obj进行Integer的相关操作。
    System.out.print(obj + " ");
});

reduce

reduce，减少。对Stream流进行处理，得到一个结果值（比如sum值）。

返回一个值。


int sum = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9).reduce(0, (acc, n) -> acc + n); // 本质，具体。返回的结果赋给第一个参数

int sum2 = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9).reduce(0, Integer::sum);
// 初始会把 0传给a，然后1传给b。
int sum3 = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9).reduce(0, (a, b) -> a + b); // 抽象，初始值 和 所有元素抽象成两个元素进行操作

int max1 = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9).reduce(0, (a, b) -> a > b ? a : b); // 抽象，获取 0 和 所有元素中的最大值

int mul1 = Stream.of(1, 2, 3, 4).reduce(1, (a, b) -> a * b); // 抽象，1 和 所有元素抽象成两个元素乘积操作。

System.out.println(sum); // 45
System.out.println(sum2); // 45
System.out.println(sum3); // 45
System.out.println(max1); // 9
System.out.println(mul1); // 24


int sum_test = Stream.of(1, 2, 3, 4, 5, 6, 7, 8, 9).reduce(0, (acc, n) -> {
    // 过程显示，n为 1 2 3 4 5 6 7 8 9
    System.out.println("acc " + acc + " n " + n);
    return acc + n;
});
// 初始元素可以创建一个HashMap然后收集信息。

collect

把Stream流里的所有元素，收集 转成一个Collection。

返回一个Collection（List、Map、Set），

// List，Collectors.toList()
List<String> list = Stream.of("Apple", "Apple", null, "Pear", "Orange")
    .filter(Objects::nonNull)
    .collect(Collectors.toList());

// Set，Collectors.toSet()
Set<String> set = Stream.of("Apple", "Apple", null, "Pear", "Orange")
    .collect(Collectors.toSet());

// Map，
Map<String, String> map = Stream.of("Apple", "Apple2", null, "Pear", "Orange")
    .collect(Collectors.toMap(obj -> obj, obj -> "1"));

System.out.println("list = " + list); // list = [Apple, Apple, Pear, Orange]
System.out.println("set = " + set); // set = [null, Apple, Pear, Orange]
System.out.println("map = " + map); // map = {null=1, Apple2=1, Apple=1, Pear=1, Orange=1}
// 分组输出（key：collection）
List<String> list = Arrays.asList("Apple", "Banana", "Blackberry", "Coconut", "Avocado", "Cherry", "Apricots");
Map<String, List<String>> groups = list.stream()
    .collect(Collectors.groupingBy(s -> s.substring(0, 1), Collectors.toList()));
System.out.println(groups); // {A=[Apple, Avocado, Apricots], B=[Banana, Blackberry], C=[Coconut, Cherry]}

toArray

Stream流的各个元素，通过包装类型的new，变成数组。

返回数组。

List<String> list = List.of("Apple", "Banana", "Orange");
String[] array = list.stream().toArray(String[]::new);

allMatch & anyMatch

是否全部都匹配该规则，是否存在一个匹配该规则。

返回 bool 值

boolean b1 = Stream.of(1, 2, 3).allMatch(obj -> obj > 1);
boolean b2 = Stream.of(1, 2, 3).anyMatch(obj -> obj > 1);
System.out.println("b1 = " + b1);
System.out.println("b2 = " + b2);
