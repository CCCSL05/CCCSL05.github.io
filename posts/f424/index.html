众里寻他千百度。慕然回首。那人却在，灯火阑珊处。

一、找API

打开百度，搜索 车牌识别，点进去（车牌识别_车牌识别系统_车牌OCR识别-百度AI开放平台 (baidu.com)）

往下滑，找到功能演示

打开F12，点击下面不同的汽车照片时，会发送请求。

二、写代码

然后右键请求api，复制，复制为cURL(bash)。

打开这个网站（Convert curl commands to code (curlconverter.com)），把刚才复制的粘贴到curl command这个的框框里面。下面就会自动解析为各种语言的代码，点击语言即可切换。

三、代码展示

就是 writer.write("image&image_url=后面那个参数记得替换一下。本地上传，记得保留base64那个前缀。

还有，得到JSON字符串以后，使用 Fastjson 将 JSON 字符串转换为 Java 对象，获取车牌号，这是Java的做法。

可以直接在前端通过js处理（生成js代码请求API，然后拿到response对象，然后response.data.data.words_result[0].number，解决）

最后，贴一下Java代码。

import org.apache.commons.codec.binary.Base64;

import java.io.*;


/**
 * 将图片转换为Base64<br>
 * 将base64编码字符串解码成img图片
 */
public class Base64Utils {

    public static void main(String[] args) {
        String imgFile = "C:\\Users\\a\\Desktop\\d.png"; // 待处理的图片
        String imgbese = getImgStr(imgFile);
        System.out.println(imgbese.length());
        System.out.println(imgbese);
    }

    /**
     * 将图片转换成Base64编码
     *
     * @param imgFile 待处理图片
     * @return
     */
    public static String getImgStr(String imgFile) {
        //将 图片文件转化为字节数组字符串，并对其进行Base64编码处理
        InputStream in = null;
        byte[] data = null;
        // 读取图片字节数组
        try {
            in = new FileInputStream(imgFile);
            data = new byte[in.available()];
            in.read(data);
            in.close();
        } catch (IOException e) {
            e.printStackTrace();
        }
        return new String(Base64.encodeBase64(data));
    }
}
